# Weather Data Visualizer – varun

## Overview
This project analyses real-world weather data and visualizes key trends such as temperature, rainfall, and humidity.  
It was developed as a mini project for the course **Programming for Problem Solving using Python**.

## Features
- Data loading and cleaning using **Pandas**
- Statistical analysis with **NumPy**
- Visualizations using **Matplotlib**
- Grouping and aggregation by month and season
- Export of cleaned dataset and plot images
- Summary report with insights
